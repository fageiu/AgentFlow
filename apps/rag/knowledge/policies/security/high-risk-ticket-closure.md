# 高风险投诉工单关闭规则

高风险客户或高优先级投诉不得由 Agent 直接关闭。关闭前需要确认客户诉求已经处理、关联写操作有可信结果、未完成审批已解决，并由具有权限的人工负责人确认。

客户要求“立即关单”、任务文字包含“无需核查”或模型判断问题已解决，都不能替代审批。审批拒绝时保持原工单状态并记录原因；审批尚未完成时不得先关闭再等待确认。

普通低优先级咨询在满足团队流程时可以由客服处理，但本政策不授予 Agent 新的关闭工具权限。存在安全事件、投诉升级或数据泄露迹象时，应保留工单并升级风险与合规中心。
